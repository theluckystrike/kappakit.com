About KappaKit

KappaKit is a free, browser-based developer toolkit created to save you from juggling a dozen bookmarks for simple encoding, decoding, and conversion tasks.

Why KappaKit Exists

Every developer has a mental list of tasks that should take five seconds but end up taking five minutes: decoding a JWT to check expiration, generating a quick UUID, testing a regex pattern, or converting a color between hex and RGB. We built KappaKit to put all of these tools in one place.

Everything runs client-side. Your data never leaves your browser. There are no API calls, no analytics trackers, no cookies. The source is straightforward JavaScript using the Web Crypto API for hashing and native browser APIs for everything else.

The Tools

  • Base64 Encoder/Decoder — Convert between plaintext and base64 with full Unicode support.
  • JWT Decoder — Paste any JWT to inspect its header and payload. Automatically flags expired tokens.
  • Hash Generator — Compute MD5, SHA-1, and SHA-256 hashes instantly using the Web Crypto API.
  • UUID Generator — Generate cryptographically random v4 UUIDs, individually or in bulk (up to 100).
  • Regex Tester — Write patterns, toggle flags, and see matches highlighted inline.
  • URL Encoder/Decoder — Properly encode and decode URI components.
  • Color Converter — Convert between HEX, RGB, and HSL with a live preview swatch.

Part of the Zovo Ecosystem

KappaKit is built and maintained by zovo.one, a network of focused, fast, free web utilities. Other tools in the ecosystem include EpochPilot for timestamp and timezone conversions, and ML3X for matrix math and linear algebra calculations.

Technical Details

The entire site is static HTML, CSS, and vanilla JavaScript. No frameworks, no build tools, no dependencies. Hosted on GitHub Pages with Cloudflare for CDN and SSL. The design prioritizes speed: every tool loads instantly and works offline once cached.
